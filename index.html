Koishi

不仅仅是一个 QQ 机器人

快速上手 →

开箱即用

高度便利的 CLI 和 API 可以让你无需基础在几分钟之内搭建自己的机器人。

功能强大

监听器，发送器，中间件，指令系统，插件系统，数据库，多机器人……它们可以让你顺利实现任何需求。

生态丰富

官方提供了大量插件和解决方案，在满足各种需求的同时，也为编写插件提供了绝佳的范例。

注意

这里是正在施工的 koishi v2 的文档。要查看 v1 版本的文档，请前往这里。

命令行
# 进入文件夹
cd my-bot
# 安装 Koishi
npm i koishi -g
# 初始化配置文件
koishi init
# 运行你的 Bot
koishi start

现在可以对你的机器人说话了：

聊天记录
A
Alice
echo 你好
Koishi
你好

提示

Koishi v2 要求您的 Node.js 的版本不小于 12，CQHTTP 的版本不小于 4.0。如果使用 WebSocket，Koishi 要求 CQHTTP 版本不小于 4.6。
